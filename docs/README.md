# newplan-prototipo

Portal Saúde Mental NR1 - Protótipo de análise psicossocial com automação de questionários Ergos, consolidação de dados e geração de documentos NR1/NR17.

## Objetivo

Desenvolver uma plataforma independente para migração, análise e geração documental focada em avaliação psicossocial.

## Estrutura

- Backend em Python (FastAPI, Pandas, geração de PDF)
- Frontend em React + TypeScript + Tailwind
- CI/CD com GitHub Actions

## Como contribuir

Clone o repositório, configure o ambiente Python e Node, e siga os padrões de commit em `contributing.md` (em desenvolvimento).
